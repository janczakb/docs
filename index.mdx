## Why Filament Short URL?

Filament Short URL is a self-hosted alternative to Bitly, Dub.co, and Rebrandly. You own your data, have no click limits, and pay $0 per month. It ships as a single Composer package that registers itself into any existing Filament v5 panel.

## Get started in four steps

<Steps>
  <Step title="Install via Composer">
    Add the package to your Laravel application:
    ```bash
    composer require janczakb/filament-short-url
    ```
  </Step>
  <Step title="Publish and run migrations">
    Publish the migration files and apply them to your database:
    ```bash
    php artisan vendor:publish --tag=filament-short-url-migrations
    php artisan migrate
    ```
  </Step>
  <Step title="Register the plugin">
    Add the plugin to your Filament Panel Provider:
    ```php
    use Bjanczak\FilamentShortUrl\FilamentShortUrlPlugin;

    public function panel(Panel $panel): Panel
    {
        return $panel->plugins([
            FilamentShortUrlPlugin::make(),
        ]);
    }
    ```
  </Step>
  <Step title="Create your first short link">
    Open your Filament admin panel, navigate to **Short URLs**, and click **Create**. Enter your destination URL and save — your first short link is live.
  </Step>
</Steps>
